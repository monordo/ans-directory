👮Authentification

Pour authentifier vos requêtes, vous aurez besoin d’une clé API.

Pour son utilisation, vous devez l’injecter dans le header de vos requêtes avec la api-key

🙇‍♀️Permissions

Ici vous pouvez trouver toutes les permissions disponibles sur l’API.

Pour la gestion des permissions nous utilisons notre propre librairie : @open-monordo/bitfield-permission-manager
